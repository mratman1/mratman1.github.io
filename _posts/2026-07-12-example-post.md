This is a placeholder post so you can see how the Maths section works.

Write your post content here using normal Markdown. LaTeX is rendered automatically
via MathJax — inline math like $e^{i\pi} + 1 = 0$ works, and so does display math:

$$
\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}
$$

Once you're happy with the layout, either edit this file in place or delete it
and add your own files to `_posts/`.

Each new post just needs a filename like `YYYY-MM-DD-title.md` and front matter
with a `title`, a `date`, and `categories: maths` — it will automatically show
up on the [Maths](/maths/) page, newest first.
